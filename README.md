# Pinterest → Email Bot

בוט שבודק את פינטרסט אוטומטית ושולח לך תמונות חדשות מהבורד ישירות למייל כקובץ מצורף.
רץ בענן (GitHub Actions) — המחשב שלך לא צריך להיות דלוק בכלל.

## הגדרה (חד פעמי, כ-5 דקות)

### 1. ליצור Repository ב-GitHub
1. תיכנס ל-github.com ותתחבר
2. לחץ על "New repository" (הכפתור הירוק)
3. תן שם, למשל `pinterest-emailer`
4. סמן "Private" (כדי שאף אחד לא יראה את זה)
5. Create repository

### 2. להעלות את הקבצים האלה
בעמוד הריפו החדש, לחץ "uploading an existing file" וגרור לתוכו את כל התיקייה
(כולל תיקיית `.github` עם התיקייה `workflows` בפנים - חשוב לשמור על המבנה).

### 3. להגדיר Secrets (כדי שהסיסמה שלך לא תופיע בקוד)
בריפו: Settings → Secrets and variables → Actions → New repository secret.
תוסיף את ארבעת אלה, אחד-אחד:

| שם ה-Secret     | ערך                                                  |
|-----------------|-------------------------------------------------------|
| `RSS_URL`       | `https://www.pinterest.com/ghruobah/n.rss`            |
| `FROM_EMAIL`    | כתובת ה-Gmail שממנה נשלח                              |
| `APP_PASSWORD`  | ה-App Password בן 16 התווים שיצרת ב-Gmail             |
| `TO_EMAIL`      | הכתובת שאליה אתה רוצה לקבל את התמונות (יכולה להיות אותה כתובת)|

### 4. להפעיל
לך ל-tab בשם "Actions" בריפו → תבחר "Check Pinterest board and email new pins"
→ יופיע כפתור "Run workflow" (מימין) → לחץ עליו כדי להריץ פעם ראשונה ידנית ולוודא שהכל עובד.

מרגע זה, זה ירוץ **לבד** כל 30 דקות ויבדוק אם יש פינים חדשים בבורד.

## איך זה עובד
- כל הרצה בודקת את ה-RSS של הבורד
- פין שכבר נשלח נשמר ברשימה (`seen_pins.json`) כדי שלא יישלח שוב
- פין חדש → התמונה מורדת בזיכרון של שרתי GitHub (לא במחשב שלך) → נשלחת אליך במייל כקובץ מצורף

## שינוי תדירות הבדיקה
בקובץ `.github/workflows/check_pins.yml`, השורה:
```
- cron: "*/30 * * * *"
```
אפשר לשנות ל-`*/15` (כל 15 דק'), `0 * * * *` (כל שעה), וכו'.

## בעיות נפוצות
- **לא מגיע מייל בכלל**: תבדוק ב-Actions tab אם ההרצה נכשלה (X אדום) ותפתח אותה לראות את השגיאה.
- **"RSS_URL" חוזר ריק**: ודא שהבורד עדיין ציבורי (לא Secret) בהגדרות הבורד בפינטרסט.
- **שגיאת התחברות ל-Gmail**: ודא שה-2-Step Verification מופעל וש-App Password נוצר נכון (לא הסיסמה הרגילה שלך).
